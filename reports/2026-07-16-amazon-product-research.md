## Do Not Pursue (Rejected Today)

### A) Fitted bed sheets (Sábana bajera ajustable, algodón)
- **Why rejected:** Size complexity is fatal for a first-launch private label — Spanish mattress widths span 80 / 90 / 105 / 135 / 150 / 160 / 180 cm, and each width needs at least two depths (25 / 30 cm). That is 10–14 SKUs before any colour variation. High return risk from mis-sized deliveries. Category is anchored by well-known bedding brands with 10k+ reviews. Margin is thin at Amazon.es price bands (€12–€22 for a single fitted sheet). Config §"Products to Avoid" flags "products with too many sizes, colours, or variations" — this is the textbook example.

### B) Duvet cover set (Funda nórdica juego, algodón percal)
- **Why rejected:** Combines every risk Rosanube should avoid at this stage — bulky (higher FBA size tier), high size variation (funda nórdica de 90 / 135 / 150 / 180 / 200 cm), high aesthetic-return risk (colour / feel / thread-count expectations), and a category dominated by large European bedding brands with heavy PPC budgets. Photography requires styled bedroom sets. Reviews are emotional and unforgiving. Config §"Products to Avoid" — bulky, brand-dominated, many variations, high return risk — all apply.

### C) Heated / electric throw blanket (Manta eléctrica calefactable)
- **Why rejected:** This is an electronic / electrical product, not a textile. Config §"Products to Avoid" explicitly excludes electronics and batteries, and this SKU adds mandatory low-voltage / CE-EMC compliance, controller-unit warranty exposure, fire-safety claims risk, and a completely different returns and support profile. Not a Rosanube fit at any risk level.

- **Textile Regulation EU 1007/2011** — fibre composition label required on all 5 SKUs (napkins, aprons, cloths, ironing cover, muslin throw); label in Spanish + Portuguese + English.
